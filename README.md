
**Library Modules**
The library is divided into **Modules**. This makes it easy to use and update.


**üì• Input Module (`read_lib`)**
       This module gets data from the user and checks if it is correct.
**Available Functions:** 
       **`read_positive_number()`**: It reads a positive number. If the input is wrong, it asks the user to try again.
       **`read_password()`**: It reads a password from the user, consisting of 3 uppercase letters between AAA and ZZZ (example: password = "AYC").

**‚úÖ Check Module (`check_lib`)**
       This module tests the data to see if it follows specific rules.
**Available Functions:**
       **`check_prime_number()`**: It checks if the number is a **Prime Number**.
       **`is_perfect_number()`**: It checks if the number is a **Perfect Number** (the sum of its parts equals the number).**

**üîç Check Module (`data_analyzer`)**
       Its job is to "look" inside the data and give you useful information.
**Available Functions:**
       **count_digit_frequency()**:It counts how many times a specific digit appears inside a larger number
